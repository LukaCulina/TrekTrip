**TrekTrip**

TrekTrip is a full-stack travel planning web app that allows users to create, explore, and share trip itineraries.

Users can build detailed trips by organizing days, locations, and activities, as well as browse other users’ trips, leave ratings, and share experiences. The platform focuses on making travel planning more interactive and community-driven.

---

**Tech Stack**

Frontend

- React
- JavaScript
- CSS / Material UI

Backend

- Spring Boot
- Spring Security (JWT authentication)
- REST API

Database

- PostgreSQL (Neon)

Deployment

- Frontend: Vercel
- Backend: Koyeb

---

**Features**

- User authentication (JWT-based login & registration)
- Create and manage trips
- Add locations and daily plans
- Rate and comment on trips
- User profiles
